Observing the Relation of SNPs to Disease¶

Using machine learning, we will determine the relationship of Single Nucleotide Polymorphism (SNP) to disease. This report specifies respective model accuracy in addition to providing insight for further genomic analysis. We use random forests, logistic regression, support vector machines, and neural networks to classify the presence of disease based on the number of mutated alleles found in 10 distinct positions.

Data Preprocessing¶

The .arff is loaded into a data set using scipy's loadarff module. It is then split into a training and target (or testing). The training set is later combined into a pandas dataframe. The data is converted to int64s and NA values are removed. We describe each variable in addition to displaying their histogram.

Model Comparisons¶

For each model, we will predict the presence of the disease using a binary classification method that incorporating all SNP predictors. Their respective performance is evaulated using auc scoring, observing its confusion matrix, and mean accuracy.

Random Forests¶

The random forests model provides a mean classification accuracy of 75%. Based off the confusion matrix, the model's ability to recall (sensitivity) is 82%. It has a precision accuracy of 73% and a specificity of 78%. After ranking the feature importance of all 10 gene locations, location 5 shows the highest relative importance at 22%. This is followed by location 10 with 20% relative importance. The ROC curve plotted below has an AUC of 81%.

# no need to permute data since random forest takes random samples

Logistic Regression¶

The logistic regression model provides a mean classification accuracy of 61.3%. Based off the confusion matrix, the model's ability to recall (sensitivity) is 84%. It has a precision accuracy of 58% and a specificity of 35%. The ROC curve plotted below has an AUC of 54%. While the model has a lower accuracy than that of a random forest model, it has a significantly higher ability to recall true positive values. This tradeoff in error may be preferred depending on the domain of the analysis. Here, we may not care for the model because geneticists would rather have an more accurate reading of precision.

Support Vector Machine¶

The SVM models are run using 2 different kernels: RBF (radial basis function) and the polynomial kernel.

RBF Kernel¶

The RBF Kernel model provides a mean classification accuracy of 80%, the highest of all models so far. Based off the confusion matrix, the model's ability to recall (sensitivity) is 95%. It has a precision accuracy of 74% and a specificity of 64%. The ROC curve plotted below has an AUC of 83%, the highest AUC so far. After using a cross-validation grid search to find the best gamma and C parameters, the best model has a C = 10 and gamma = 0.1.

Neural Network using Weka¶

The multiple layer perceptron algorithm in Weka was used to classify the 10 features. Attributes used:

1. Number of Hidden Layers: 3
2. Momentum: 0.2
3. Learning Rate: 0.3
4. Training Time: 500

Given these parameters, the accuracy and sensitivity was 80%. The fall-out rate was 21%, precision 83%, and recall at 80%. It's AUC is the second highest, 81%. However, I would say this is strongest, most balanced model. This neural network and SVM (RBF) model provide the best accuracy in classifying disease as a function of allele mutations.
